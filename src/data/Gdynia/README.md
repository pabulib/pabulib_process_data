# Gdynia

Source data for Gdynia is publicly available online.

Main sources:

- Open data portal: https://otwartedane.gdynia.pl/
- BO website archive: https://bo.gdynia.pl/poprzednie/

Year-specific links used in this repository:

- 2025:
  - open data dataset: https://otwartedane.gdynia.pl/pl/dataset/budzet-obywatelski-2025
  - results PDF: https://bo.gdynia.pl/wp-content/uploads/2025/09/wyniki-glosowania.pdf
- 2024:
  - open data dataset: https://otwartedane.gdynia.pl/pl/dataset/budzet-obywatelski-2024
  - results PDF: https://bo.gdynia.pl/wp-content/uploads/2024/09/wyniki-glosowania.pdf
- 2023:
  - open data dataset: https://otwartedane.gdynia.pl/pl/dataset/budzet-obywatelski-2023
  - results PDF: https://bo.gdynia.pl/wp-content/uploads/2023/09/wyniki-glosowania.pdf
- 2022:
  - open data dataset: https://otwartedane.gdynia.pl/pl/dataset/budzet-obywatelski-2022
  - results PDF: https://bo.gdynia.pl/wp-content/uploads/2022/09/wyniki-glosowania.pdf
- 2021:
  - open data dataset: https://otwartedane.gdynia.pl/pl/dataset/budzet-obywatelski-2021
  - results PDF: https://bo.gdynia.pl/wp-content/uploads/2021/12/wyniki-glosowania_wszystkie-projekty_na-strone.pdf
  - summary report: https://bo.gdynia.pl/wp-content/uploads/2021/12/2021-10-06_Raport-podsumowujacy-BO-2021_aktualizacja.pdf
- 2020:
  - BO archive page: https://bo.gdynia.pl/poprzednie/budzet-obywatelski-2020/

Notes:

- For 2021-2025, this repository uses source files published on the Gdynia open data portal together with official BO results PDFs.
- Gdynia voting was conducted online in these newer editions, so generated `.pb` files use `voting_method=internet`.
